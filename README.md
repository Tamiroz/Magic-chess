Magic Chess ♟️✨
Magic Chess is an innovative web-based chess variant designed to bridge the gap between players of different skill levels. By introducing randomized magical effects into the traditional "no-chance" environment of chess, the game creates a dynamic and unpredictable experience where strategy meets serendipity.

🌟 Concept
The core vision behind this project was to balance the odds in matches with a significant level difference. To achieve this, I engineered a logic system that triggers "chance" mechanics, forcing even experienced players to adapt their strategies on the fly.

🛠️ Key Features
Dynamic Special Effects: Includes randomized mechanics such as:

Lightning Strikes: Impacting specific squares.

Freeze: Immobilizing pieces for a set duration.

Portals: Creating non-traditional movement paths across the board.

Custom Physics Engine: Built from scratch to handle gravity and collision logic within the game environment.

Responsive UI: A clean, interactive interface built for a seamless user experience.

🚀 Tech Stack
Frontend: HTML5, CSS3, Vanilla JavaScript.

Logic: Custom game state management and randomized event triggers.

Development Tools: AI-assisted coding and iterative prototyping (originally prototyped in Python/Pygame).
